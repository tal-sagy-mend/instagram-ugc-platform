## Design System Notes

### Colors
- **Primary**: Instagram gradient (purple to pink to orange)
- **Success**: Green (#10B981) for approved permissions
- **Warning**: Yellow (#F59E0B) for pending requests
- **Error**: Red (#EF4444) for denied/expired
- **Neutral**: Gray scale for text and backgrounds

### Typography
- **Headings**: Bold, 24-32px
- **Body**: Regular, 16px
- **Small**: 14px for metadata
- **Font**: System fonts (San Francisco on Mac, Segoe UI on Windows)

### Spacing
- **Grid Gap**: 16px between items
- **Card Padding**: 16px
- **Section Spacing**: 32px vertical
- **Modal Padding**: 24px

### Components
- **Buttons**: Primary (filled), Secondary (outlined), Text (minimal)
- **Cards**: White background, subtle shadow, rounded corners (8px)
- **Inputs**: Border, focus ring, placeholder text
- **Modals**: Overlay with blur, centered, max-width 800px

## Technical Implementation Notes

### State Management
- Use React Query for server state (content, permissions)
- Use Zustand/Redux for UI state (modals, filters, selections)
- Cache API responses for 1 hour

### Performance
- Lazy load images (intersection observer)
- Virtual scrolling for large lists
- Debounce search inputs (300ms)
- Pagination/infinite scroll for results

### Accessibility
- Keyboard navigation (Tab, Enter, Escape)
- ARIA labels for screen readers
- Focus indicators
- Alt text for all images
- Color contrast ratios meet WCAG AA

### Browser Support
- Chrome/Edge (latest 2 versions)
- Firefox (latest 2 versions)
- Safari (latest 2 versions)
- Mobile Safari (iOS 14+)
- Chrome Mobile (Android 10+)
